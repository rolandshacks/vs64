
# VS64 Development Tools Bundle []()

The VS64 Package Manager automatically downloads and installs bundled development tools.

- Acme, KickAssembler
- Vice, X16Emulator
- LLVM-MOS, Oscar64, CC65
- Java Runtime Environment
- Exomizer

Please note: not all development bundles are available for each platform.
For platforms not currently supported by the package manager, please
manually install and configure using the VS64 settings pages.
